# scroll background with speed

Sets the background image to scroll at a set speed. The background will loop when the scroll reaches the edges of the image.

```sig
scroller.scrollBackgroundWithSpeed(50, 0)
```

## Parameters

* **vx**: The speed to scroll on the horizontal axis
* **vy**: The speed to scroll on the vertical axis
* **layer**: Chooses the parallax layer this operation applies to. Defaults to 0

## Example #example
