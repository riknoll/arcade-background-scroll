# set layer image

Sets the image for one layer in the parallax scrolling stack. Parallax is an effect that makes it appear as though a 2D game is in 3D by scrolling different layers of the background at different speed.

All of the other blocks in this extension take in a "layer" parameter that can be used to control which layer they apply to.

```sig
scroller.setLayerImage(scroller._backgroundLayer(scroller.BackgroundLayer.Layer0), img`.`)
```

## Parameters

* **layer**: Chooses the parallax layer to set the image for
* **image**: The image to set for that layer. Should be at least the size of the screen (160 x 120)
